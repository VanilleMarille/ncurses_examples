# ncurses example - Detect Arrow key + Draw box

* Key UP = int 2
* Key DOWN = int 3
* Key RIGHT = int 4
* Key LEFT = int 5
* Keys Other = int 1

drawbox(x,y,width,height)

Compile with: gcc example.c -o example -lncurses
